# 스크린 G핸디 계산기

골프존식 G핸디 기준으로 개인전/팀전 순위를 계산하는 무료 웹 계산기입니다.

## 주요 기능

- 플레이어 등록
- G핸디캡 입력
- 개인전/팀전 선택
- 2~4팀 구성
- 중복 팀 선택 방지
- 최종 스코어 입력
- 개인 순위 계산
- 팀 순위 계산
- 플레이어 이름/G핸디 브라우저 저장

## 계산식

```text
최종 결과 = 최종 스코어 - G핸디캡
```

값이 낮을수록 순위가 높습니다.

## 입력 예시

```text
G핸디캡 -7 / 최종 스코어 +5
= +5 - (-7)
= 12개 오버
```

```text
G핸디캡 -7 / 최종 스코어 -10
= -10 - (-7)
= 3개 언더
```

## 주의사항

이 앱은 72타 기준 필드 총타수 계산기가 아닙니다.

골프존식 `+5`, `-3`, `0` 결과 입력 방식 기준입니다.

## 로컬 저장

플레이어 이름과 G핸디는 서버가 아니라 사용자의 브라우저 `localStorage`에만 저장됩니다.

저장하지 않는 값:

- 최종 스코어
- 경기 방식
- 팀 구성
- 계산 결과

`전체 초기화`는 이번 게임 진행 상태만 초기화하고 저장된 플레이어는 유지합니다.

`저장 데이터 전체 삭제`는 `screenHandyCalculator.`로 시작하는 앱 localStorage 데이터를 모두 삭제합니다.

## GitHub Pages 배포 방법

1. GitHub 저장소에 프로젝트 파일을 업로드합니다.
2. 저장소의 `Settings` 메뉴로 이동합니다.
3. `Pages` 메뉴를 선택합니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. Branch를 `main`, 폴더를 `/root`로 선택합니다.
6. 저장 후 표시되는 GitHub Pages 주소로 접속합니다.

## GitHub CLI 자동 배포

Git과 GitHub CLI가 설치되어 있고 GitHub CLI 로그인이 완료되어 있으면 아래 명령으로 저장소 생성, 커밋, push, GitHub Pages 활성화를 자동 처리할 수 있습니다.

```powershell
.\scripts\deploy_github_pages.ps1
```

최초 1회 로그인은 직접 진행해야 합니다.

```powershell
gh auth login
gh auth setup-git
```

## 배포 전 수동 테스트 시나리오

### 테스트 1: 개인전

- 플레이어 3명 등록
- 개인전 선택
- 최종 스코어 입력
- 순위가 낮은 결과값 기준으로 정렬되는지 확인

### 테스트 2: 팀전

- 플레이어 4명 등록
- 팀전 선택
- 2팀 선택
- 각 팀에 2명씩 배정
- 중복 선택이 방지되는지 확인
- 팀 합산 결과가 정상 표시되는지 확인

### 테스트 3: 입력값

아래 값이 모두 정상 처리되는지 확인합니다.

```text
+5
5
-3
0
```
